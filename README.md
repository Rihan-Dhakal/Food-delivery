Food Delivery Web App

A dynamic food delivery web application built with HTML, CSS, and JavaScript. This project demonstrates interactive cart functionality, a responsive product carousel, and mobile-friendly navigation.

Features

Add / Remove Items: Seamlessly add products to the cart and adjust quantities.

Live Totals: Automatically updates total price and quantity.

Product Carousel: Interactive product slider powered by Swiper.js.

Mobile-Friendly Menu: Responsive hamburger menu for small screens.

Clean UI: Simple and intuitive design for easy user interaction.



Usage

Browse the products in the carousel.

Click Add to Cart to include items.

Use + / - buttons to adjust quantity.

See live updates in cart total and quantity.

Technologies Used

HTML5

CSS3

JavaScript (ES6)

Swiper.js (for carousel)

Future Improvements

Save cart data with local storage to persist across sessions.

Add checkout functionality.

Connect to a backend API for real product data and order processing.
